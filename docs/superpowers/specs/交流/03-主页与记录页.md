# 03 · 主页与记录页（文字）

## 上下文（给 Trae）
01-02 完成：IndexedDB + entries 的增删查统计都有了。现在做界面：屏1主页 + 屏2记录页（这条先只做文字输入，语音下一条）。

## 要做的
- **屏1 · 主页**：顶部标题「忘事日记」；中间一个大按钮「记一笔忘事」（主操作）；下面一个小按钮「切场检查」（这条先占位，点击先 alert 占位即可，别实现）；底部一行「本周已记 N 笔」（调 `getStats().weekCount`）。
- **屏2 · 记录页**：一个大文本框（placeholder「忘了啥？」）；下面一行标签 chips：`切换场景 / 到家 / 下车 / 出门 / 刚刷完手机`，可多选可空（选中高亮）；底部「保存」按钮。保存调 `addEntry({ text, tags })`，成功后回主页 + 轻提示「已记」。
- **页面切换**：单页应用，用 JS 切 div 的显示/隐藏（`display`），**不要**用多 html 文件、不要用路由库。
- 主页进入时刷新本周数。

## 约束
vanilla JS 手写 DOM，不引框架。样式移动优先，按钮要够大好点。

## 验收标准
1. 主页两个按钮，点「记一笔」进到记录页。
2. 输文字、选几个标签、点保存 -> 回主页，底部「本周已记」数 +1。
3. F5 刷新数据还在（走 IndexedDB，不是内存）。
4. 标签可多选、可取消选择、可全不选也能保存。
