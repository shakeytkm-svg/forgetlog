# 05 · 切场检查与看板

## 上下文（给 Trae）
04 完成：记录页能语音输入了。现在做「切场检查」功能 + 屏3看板。

## 要做的
- **切场检查**：主页「切场检查」按钮（03 里占位的那个）点击 -> 弹一个全屏遮罩，中间一句大字「带东西了吗？有要办的吗？」，底部一个「过完了」按钮关掉。纯提醒，**不存任何数据**。
- **屏3 · 看板**：主页加第三个按钮「看板」进入。看板内容：
  - 顶部大数字「本周 N 笔」（`getStats().weekCount`）；
  - 「按场景分布」：每个标签一行，标签名 + 一根横条（长度按比例）+ 数量；
  - 「按星期几分布」：周一到周日，每行一根横条 + 数量；
  - 「最近 10 条」：列表，每条显示 `时间` + `文字` + `标签`，点一条弹出二次确认后删除（调 `deleteEntry`），删完刷新看板。

## 约束
横条用 div + 宽度百分比即可，不引图表库。删除一定二次确认，别误删。

## 验收标准
1. 切场检查弹窗能开、能关，关掉后主页正常。
2. 看板能显示真实数据（先手动 `addEntry` 几条带不同标签、不同时间的测试数据）。
3. 横条长度按数量比例对得上。
4. 列表点删除有确认框，确认后该条消失、统计数字同步更新。
