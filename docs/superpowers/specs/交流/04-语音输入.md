# 04 · 语音输入

## 上下文（给 Trae）
03 完成：记录页有文本框和标签了。现在给记录页加「按住说话 -> 自动转文字」。

## 要做的
记录页文本框上方加一个「🎤 按住说话」按钮：
- 用 Web Speech API：`new (window.SpeechRecognition || window.webkitSpeechRecognition)()`。
- `lang = 'zh-CN'`、`continuous = false`、`interimResults = true`。
- **按下**开始识别，**松开**停止。
- 识别中把 `interim`（中间结果）实时显示在按钮下方一行小字。
- 最终结果**追加**进文本框：若文本框已有文字，先加个空格再拼，**不要覆盖**原内容。
- 浏览器不支持时（`SpeechRecognition` 不存在，如部分 iOS）：**隐藏**这个按钮，并显示一行小字「本设备不支持语音，可手打」，**不要报错崩页面**。

## 约束
只改记录页相关文件，别动数据层。注意释放识别对象（`rec.stop()`），松开要真停。

## 验收标准
1. 按住按钮说话，能把中文转成文字（比如「我忘了带蛋糕」），松开后文字进文本框。
2. 说话过程中能看到中间结果实时变化。
3. 连说两句（先存一句、再说一句），第二句是追加、不是覆盖。
4. 在不支持的浏览器上打开，按钮隐藏、有提示、页面不报错。
